Now imagine having a computer with just one switch.
It could only represent two numbers.

`0` (off)
`1` (on).

But what if we add another switch?
Now we have four combinations, meaning we can represent four numbers:

`00` (both off) = 0
`01` (first off, second on) = 1
`10` (first on, second off) = 2
`11` (both on)  = 3
