Each transistor can hold a bit. Short for binary digit. Bi as in two. Digit as in a single number.

Imagine having a computer with just one bit.
It could only represent two numbers.

`0` (off)
`1` (on).

What if we add another bit?
Now we have four combinations, meaning we can represent four numbers:

`00` (both off) = 0
`01` (first off, second on) = 1
`10` (first on, second off) = 2
`11` (both on)  = 3
