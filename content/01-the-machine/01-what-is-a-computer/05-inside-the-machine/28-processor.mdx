A processor is a chip that does things with bits.'A processor is billions of switches, organized into circuits, coordinated by a clock, to execute instructions.

It takes bits in, **performs an operation** — adds them, compares them, moves them around — and puts new bits out.

Do this **billions of times a second** and you can run any program ever written.

Your computer has more than one processor.

Each one built for a different kind of work.
