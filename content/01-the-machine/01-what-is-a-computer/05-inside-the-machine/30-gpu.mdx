Graphics Processing Unit. In common terms known as the graphics card.

Originally built to render images on screen. Every pixel needs the same calculation done at the same time, so the GPU was built differently.

If you have a ever played a video game that was graphically intensive, you know how important a GPU is.

Not one powerful processor, but **thousands of simpler ones**, all running at the same time.

It turned out this made it perfect for something else entirely.

You will hear about this again.
