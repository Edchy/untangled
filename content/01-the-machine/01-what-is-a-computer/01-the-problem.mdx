## A computer is a machine that follows instructions.

That sounds obvious now, but it was once a strange idea. A machine could be built to do one job: grind grain, lift water, keep time. The leap was imagining a machine whose job was not one task, but many tasks.

The trick is that the task has to be written down clearly enough for the machine to follow. No guessing. No common sense. Just tiny steps, one after another.

<Callout>
The heart of computing is not electricity. It is the idea that a process can be broken into simple instructions.
</Callout>

So before we talk about AI, we start here: with a machine that can follow a recipe.
