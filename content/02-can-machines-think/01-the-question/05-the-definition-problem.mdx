The obvious move.
Define "thinking." Pin it down exactly, then test whether machines meet the standard.

Turing tried. It failed.

Any definition you write is either too wide—a thermostat "thinks" by that measure—or too narrow, excluding things humans clearly do. "Thinking" resists clean boundaries.

So Turing did something cleverer still. He abandoned definitions altogether.
