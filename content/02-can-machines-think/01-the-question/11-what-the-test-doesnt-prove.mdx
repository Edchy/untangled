Here's where we need to be honest.

Passing the Imitation Game doesn't prove a machine thinks. It proves the machine fooled someone. Those are not the same thing.

A very good actor can fool an audience into believing they're devastated, without being devastated at all. A machine that passes Turing's test might be doing something similar — generating outputs that look like thinking, without any thinking happening inside.

Turing knew this. He wasn't claiming the test proved understanding. He was claiming it was a reasonable *stand-in* for the real thing, given that we can't define the real thing.

Whether that stand-in is good enough is a question we'll come back to.
