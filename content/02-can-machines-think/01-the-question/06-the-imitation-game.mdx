Here is what Turing proposed instead.

Imagine three people in separate rooms. One is a man. One is a woman. One is a judge. The judge can only communicate with the other two by exchanging typed messages. The judge has to figure out which is which.

That's the starting point. Now Turing makes a substitution: what if one of the rooms contains a machine instead of a person? The judge still has to figure out which is which.

If a machine can play this game as well as a person — if a judge can't reliably tell the difference — should we say the machine is thinking?

Turing's answer: probably yes.
