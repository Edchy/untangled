Turing began with a game involving three people: a man, a woman, and an interrogator. 
The interrogator’s goal was to figure out which player was the woman, using only typed messages. The man tried to mislead the interrogator, while the woman tried to help identify herself honestly.

Turing then asked: what happens if a machine takes the man’s place? If the interrogator can’t reliably tell the machine from a human, should we say the machine is thinking? That is the basic idea behind Turing’s argument.
