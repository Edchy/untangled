Let's be precise about what the Logic Theorist did — because it matters.

It didn't understand mathematics. It didn't know what a theorem *meant*. It had a set of rules: if you have an expression of this shape, you can transform it into an expression of that shape. It tried combinations of those rules, checking each result to see if it matched the target theorem.

When it succeeded, it wasn't because it had insight. It was because it had searched through enough combinations to find one that worked.

This is still impressive. Searching through combinations is hard. But it's a different thing from understanding.
